# task-manager
This project is a command-line task management system written in Python as part of a capstone assigment. It allows users to log in, manage tasks and generate reports using text files.

## Features
- User authentication using a user.txt file
- Register new users (admin only)
- Add tasks and assign them to existing users
- View all tasks or only tasks assigned to the logged-in user
- Edit tasks or mark them as complete
- View completed tasks
- Delete tasks by title
- Generate and display task and user statistics reports (admin only)

## Files used/generated
- user.txt – stores usernames and passwords
- tasks.txt – stores all task data
- task_overview.txt – generated task statistics
- user_overview.txt – generated user statistics

## Requirements
Python 3.x
No external libraries required
